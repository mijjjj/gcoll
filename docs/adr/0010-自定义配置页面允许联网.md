# ADR-0010 自定义配置页面允许联网

## 状态

已采纳。

## 背景

插件自定义配置页面可能需要访问在线文档、在线模板、云端配置服务或第三方系统。完全禁止联网会限制插件体验和扩展空间。

## 决策

插件自定义配置页面允许联网，安全性由用户负责。

框架仍负责：

- 插件包校验。
- 插件权限展示。
- 配置保存和服务端校验。
- 敏感字段脱敏。
- 配置审计。

## 影响

正面影响：

- 插件配置页面能力更灵活。
- 更容易接入第三方在线服务。

负面影响：

- 安全风险增加。
- 企业内网环境需要额外网络治理策略。

## 后续动作

- 在插件安装和配置页面展示联网风险提示。
- 在插件权限中保留网络访问声明。
- 文档中说明联网风险由用户负责。
