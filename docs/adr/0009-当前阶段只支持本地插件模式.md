# ADR-0009 当前阶段只支持本地插件模式

## 状态

已采纳。

## 背景

在线应用商城涉及远程插件源、下载、审核、签名分发、版本推荐和安全治理。当前阶段更需要先验证插件包、本地安装、设备配置、运行和调试能力。

## 决策

当前阶段只支持本地插件模式：

- 插件包由用户本地导入。
- 宿主执行本地插件安装、启用、禁用、启动、停止、升级和回滚。
- 应用商城只预留接口扩展。
- 在线商城和远程插件源后续再完善。

## 影响

正面影响：

- MVP 范围更清晰。
- 降低联网分发和审核复杂度。
- 更快验证插件运行时。

负面影响：

- 首版没有在线插件发现和远程安装体验。
- 插件分发需要用户手动管理。

## 后续动作

- 保留 marketplace 模块边界。
- 本地插件包格式和签名校验仍按正式机制设计。
- 后续在不破坏本地插件能力的前提下扩展在线商城。
